# NCA Fundraising Performance & Planning Assistant

**Development & Fundraising Department — Northbridge Community Alliance**  
GBA 479 Take Home Final · Simon Business School · 2026

---

## What It Does

A conversational revenue performance assistant for the NCA Development & Fundraising team. Staff ask plain-language questions about monthly revenue, year-over-year trends, cost effectiveness, and goal pacing — the system runs deterministic financial calculations and returns cited, number-backed answers. Includes what-if scenario modelling.

---

## Architecture

```
User Question
     │
     ▼
┌─────────────────────────────────┐
│  ROUTER  [LLM — Haiku]          │  Classifies intent →
│  route_question()               │  snapshot / yoy / cost_kpi /
└──────────────┬──────────────────┘  pacing / what_if / multi / out_of_scope
               │
               ▼
┌─────────────────────────────────┐
│  TOOL LAYER  [Python]           │  5 deterministic calculation
│  get_fundraising_snapshot()     │  functions — every dollar
│  compare_yoy()                  │  amount computed here,
│  cost_effectiveness()           │  never by the LLM
│  goal_pacing()                  │
│  what_if_scenario()             │
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│  GENERATOR  [LLM — Opus]        │  Interprets tool output →
│                                 │  clear business language
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│  VALIDATOR  [Python]            │  Checks: cited figures present +
│  validate_response()            │  data source line — regenerate
└─────────────────────────────────┘  once if either missing
```

**Task decomposition:**
- **Python (deterministic):** All financial calculations — MTD/YTD/TTM aggregation, YoY comparison, cost-to-raise KPI, pacing math, scenario projection
- **LLM — Haiku:** Intent routing (fast, low-cost)
- **LLM — Opus:** Natural language interpretation of tool output

## Financial Tools

| Tool | What It Computes |
|------|-----------------|
| `get_fundraising_snapshot` | MTD revenue/costs/net + YTD + TTM + revenue breakdown by stream |
| `compare_yoy` | Any metric vs same period last year (month + YTD) |
| `cost_effectiveness` | Cost-to-raise-$1 and net margin over trailing window |
| `goal_pacing` | YTD raised vs annual goal, required run-rate, pace status |
| `what_if_scenario` | Adjusted net/margin if revenue or costs shift by % |

---

## Key Design Decisions

**LLM interprets; Python calculates.** Every dollar amount and percentage in every response is computed by deterministic Python functions — not generated by the LLM. The LLM's only job is turning structured JSON tool output into clear business language.

**Fail-closed financial validation.** The validator checks that every response cites specific figures and includes a data source line. If either is missing, the system regenerates — preventing vague summaries that could mislead leadership preparing board updates.

**What-if as deterministic math.** Scenario analysis (e.g. "+10% revenue, -5% costs") is a Python calculation, not LLM speculation. Projections are fully reproducible and auditable.

## Example Interactions

```
You: Give me a quick revenue snapshot for October 2025

→ Route: snapshot | Tool: get_fundraising_snapshot(as_of="2025-10")

October 2025 revenue was $101,666.55 with net income of $78,610.52.
YTD through October: $1,058,865.40 raised against $271,983.65 in costs,
for a YTD net of $786,881.75...

📊 Data source: Northbridge fundraising history (January 2021 – December 2025)
```

```
You: We have a $1.2M annual goal. Are we on track as of October?

→ Route: pacing | Tool: goal_pacing(as_of="2025-10", annual_goal=1200000)

Status: AHEAD — $1,058,865 raised YTD against a $1.2M goal.
$141,135 remaining with 2 months left. Required average: $70,567/month.
Current YTD monthly average: $105,887 — running 50% above required pace...
```

```
You: What if we increase revenue by 10% and cut costs by 5%?

→ Route: what_if | Tool: what_if_scenario(revenue_adj=0.10, cost_adj=-0.05)

Baseline TTM net: $1,003,044 (74.8% margin)
Adjusted net: $1,153,993 (78.3% margin) — +$150,949 improvement...
```
